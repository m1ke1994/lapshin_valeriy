## CMS: как редактировать контент

1. Поднимите бэкенд: `cd backend && python manage.py migrate && python manage.py seed_cms && python manage.py runserver`.
2. Зайдите в админку: `http://localhost:8000/admin/` (нужен superuser).
3. Основные модели:
   - **Общие настройки** — название сайта и кнопка в шапке.
   - **Разделы сайта** — каждый блок (Хиро, Ценности, Подход, Компетенции, Доверие, Проекты, Сертификаты, Контакты). Тексты RU/EN рядом. Порядок задаётся числом, можно выключить секцию.
   - **Пункты списков** — строки внутри секции (например, пункты ценностей или шаги подхода). Редактируются прямо внутри секции (inline).
   - **Проекты** — карточки проектов. Заполняйте RU/EN тексты, теги через запятую.
   - **Сертификаты** — карточки сертификатов/наград. Картинку можно загрузить, файл PDF опционально.
   - **Заявки** — входящие заявки с формы (read-only).
4. Фронтенд берёт контент с `/api/site/`. Если API недоступен, используется встроенный дефолт из Nuxt.
5. Локаль выбирается только на фронте: поля `_ru` и `_en` отдаются одновременно, сайт сам выбирает.
